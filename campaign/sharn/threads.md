# threads.md — žive niti / otvoreni plotovi

Jedno mjesto za sve otvorene niti, kuke i questove u igri. **Ovo su potencijali, ne razriješene priče.** Nit živi dok je otvorena; kad se zatvori (za stolom), premjesti je u `log/` kao činjenicu.

Format po niti:
```
- [otvorena/u toku/zatvorena] <kratak naziv niti>
  - Izvor: (PC nit / frakcija / NPC / situacija)
  - Trenutno: (gdje stoji, šta je poznato)
  - Moguće kuke: (kako se MOŽE dotaći dalje — bez propisanog ishoda)
```

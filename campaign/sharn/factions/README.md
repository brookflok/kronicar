# factions/ — tracker velikih sila (fronts)

Velike sile Sharna kao **fronts** (iz Kroničar metode, DIO A). Ovo je motor "situacija u pokretu": sile se kreću svojim ciljevima bez obzira na party, a iz toga izviru situacije. Po jedan fajl po frakciji (`<frakcija>.md`) ili `factions.md` za pregled.

**Grim portents su potencijalni koraci, ne raspored.** Pratim koliko su se približili cilju kroz odigrane sesije.

Kandidati su već razrađeni u `settings/sharn/03_master/cross_references.md` (Dreaming Dark, Lords of Dust, Boromar↔Daask rat, Aurum, zavjera o monarhiji, dragonmarked houses). Ovdje držim njihovo ŽIVO stanje za ovu kampanju; ne prepisujem lore, samo napredak i odnos prema party-ju.

## Template (`<frakcija>.md`)
```
# <Frakcija>  (front)

- Goal: (šta žele postići)
- Odnos prema party-ju: (ne znaju za njih / rivali / lov / saveznici...)
- Ključni NPC-jevi: (referenca na settings master / npcs/)

## Grim portents (potencijalni koraci napretka)
1. [ ] ...
2. [ ] ...
3. [ ] ...
( čekiraj kad se desi za stolom; dodaj nove kad stari padnu )

## Trenutno stanje  [DM]
- (gdje su sada; šta rade kao reakcija na zadnje poteze likova)
```

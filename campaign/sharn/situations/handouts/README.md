# handouts/: propovi za igrače

Materijal koji se **pokazuje igračima** tokom istrage: novinski izresci, pisma, dokumenti, pozivnice, kartice predmeta. Sve je pisano in-world (glas svijeta, bez [DM] bilješki) i može se odštampati ili poslati kao slika za stolom. Za PDF se može iskoristiti isti alat kao za `party/handouts/` (`make_pdf.py`).

Skriveni konac kroz handoute je **split-ring sigil** (krug presječen pravom crtom). Pojavljuje se na više propova iz različitih slučajeva. Ne objašnjavaj ga; pusti igrače da sami primijete da se ponavlja. Puno objašnjenje spleta je u [[_braid]] [DM].

# party/ — igrači i likovi (PC)

Po jedan fajl po liku (`<pc-ime>.md`). Plus opciono `party.md` za stvari na nivou grupe (level, sredstva, zajednički ciljevi, gdje su trenutno). Ovdje držim ono što je iz RotLDM korak 1 (Review the Characters): da bih ostatak pripreme gradio oko likova.

**Lične kuke su potencijal, ne razriješena priča.** Pišem otvorene niti koje se MOGU dotaći, ne ono što će se desiti.

## Template za PC (`<pc-ime>.md`)
```
# <PC ime>  ( igrač: <ime igrača> )

- Rasa / klasa / level:
- Background (Eberron):
- Motivacije i ciljevi:
- Bonds (veze s drugim PC-jevima / frakcijama / NPC-jevima):
- Magic-item wishlist (tipovi koje igrač voli):
- Veza sa Sharnom (odakle, koga zna, koji district):

## Lične niti (potencijal, [DM])
- (otvorene kuke vezane za ovog lika — npr. "nestali rođak u Lower Duri", bez razrješenja)

## Stanje / bilješke
- (zadnje bitno; mijenja se po sesiji)
```

# world/ — stanje svijeta i player-facing canon

Ovdje je zajednička slika svijeta koju **igrači već imaju** (i sve buduće slične handoute o svijetu).

- **`Stanje Khorvairea.md`** — brifing o cijelom Khorvaireu na **998 YK** koji je DM poslao igračima. To je **[PLAYER]** baseline: šta svako iole obrazovan zna o nacijama, dragonmarked kućama, kriminalnim organizacijama, goblinima/Dhakaani, inženjerskim guildovima, i vijestima zadnjih 6 mjeseci.
- **`_sistematizacija_dm.md`** — [DM] digest: izvlači ono što je bitno za NAŠU kampanju (frakcije, kuke iz vijesti, šta nadograđuje/mijenja 3.5 Sharn knjigu), i pokazuje gdje to ide u našoj strukturi.

## Odnos prema ostalom
- `settings/sharn/` = 3.5 knjiga (Sharn lore, statičan kanon).
- `world/Stanje Khorvairea.md` = 998 YK, 5e-era, player-facing; **nadograđuje i osvježava** Sharn knjigu (npr. Cannith podijeljen na tri ogranka, aktuelni vladari, Day of Mourning posljedice, Pilfered Wand banda). Kad se 3.5 knjiga i ovaj brifing razilaze oko aktuelnog stanja, **vrijedi ovaj brifing** (to je ono što igrači znaju).
- `factions/`, `situations/`, `threads/` = živo stanje izvedeno odavde + iz party fajlova.

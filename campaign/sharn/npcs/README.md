# npcs/ — NPC-jevi na tabli

NPC-jevi koje je party stvarno upoznao ili koji su bitni za OVU kampanju. Razlikuje se od knjiškog rostera u `settings/sharn/` (gdje su klase, agende i lokacije). Ovdje pratim **trenutni odnos prema party-ju** i stanje. Po jedan fajl po NPC-u (`<ime>.md`), ili `npcs.md` lista za manje bitne.

Ako NPC već postoji u Sharn masteru, referenciram ga (ne prepisujem statove) i dodajem samo ono što se mijenja kroz igru.

## Template (`<ime>.md`)
```
# <NPC ime>

- Iz Sharn mastera? (da → koji district/frakcija; ne → [HOMEBREW])
- Rasa / klasa (ako postoji u izvoru):
- Veza sa party-jem: (kako su se sreli, čemu služi)
- Character archetype (za roleplay):
- Trenutni odnos / attitude: (Hostile/Unfriendly/Indifferent/Friendly/Helpful)
- Šta zna / šta želi sada:
- Jedna tajna  [DM]:

## Stanje / zadnji kontakt
- (mijenja se po sesiji)
```

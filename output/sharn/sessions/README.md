# sessions/ — gotove sesije

Ovdje idu sesije spremne za sto, po redu (`session_00_...`, `session_01_...`). Svaka se sklapa Kroničar metodom (`method/03_method/kronicar_method.md`, DIO F) iz banke `campaign/sharn/situations/` + stanja u `campaign/sharn/` (party, factions, threads).

- **`session_00_session_zero.md`** — radna checklista za Session 0 (dogovor o kampanji, stolu i likovima; popunjava se sa igračima).

Nakon što se sesija odigra, kratak sažetak ide u `campaign/sharn/log/`, a posljedice u `factions/` i `threads.md`.

# method/ — Kroničar metoda

Ovdje žive knjige o GM zanatu (pisanje i vođenje sesija/kampanja) i ujedinjena metoda izvedena iz njih. Metoda je univerzalna, ne vezana za jedan setting.

## Pipeline
`00_source/` (PDF-ovi) → `01_batches/` (tekst po poglavlju) → `02_summaries/` (vjeran sažetak, drugi prolaz provjere) → `03_method/` (master metode).

## Master metode (`03_method/`)
- `kronicar_method.md` — JEDNA ujedinjena metoda: korak-po-korak prep proces od ideje do gotove sesije/kampanje. Svaki korak tagovan: `[NEUTRAL]` (sistemski neutralno, treba prevod u D&D 3.5/Eberron) ili `[D&D]` (native).
- `method_index.md` — navigacija kroz korake i tehnike.
- `sources_traceability.md` — koji korak metode dolazi iz koje knjige i strane.

## Knjige (00_source)
- *02.04. Return of the Lazy Dungeon Master* — sistemski neutralan prep (Sly Flourish). [NEUTRAL]
- *02.02. Fantastic Locations* — D&D lokacije/enkaunteri. [D&D]
